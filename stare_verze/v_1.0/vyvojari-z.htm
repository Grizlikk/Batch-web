PRO ZAČÁTEČNÍKY

Pokud neumíš dělat webové stránky, zde najdeš vše, co je potřeba vědět, k udělání základní rozšiřující webové stránky, například s vysvětlením nějakého Batch programu


Jak funguje psaní webových stránek:

K psaní webových stránek nepotřebujete žádné speciální programy, stačí vám pouze poznámkový blok, ale v profesionálních editorech se vám bude pracovat mnohem lépe. K otevření webové stránky stačí daný soubor otevřít v libovolném webovém prohlížeči.

Všechny soubory webových stránek mají příponu .htm nebo .html (obvykle se používá .htm, protože je to prostě kratší).


Úvod do HTML:

Všechny stránky, které zde budete psát, budou v jazyce HTML. HTML není programovací jazyk, ale tzv. značkovací jazyk. To znamená, že každý "příkaz" je jakási značka, uzavřená do <>.

Každá značka má vždy nějaký začátek a nějaký konec. Ukončovací značka vždy obsahuje navíc znak "/", např: otevírací znakča: <p> a uzavírací značka: </p>


Základní značky:

Každý kód webové stránky vždy začíná s <!DOCTYPE html>. Jedná se o definici verze HTML, zde se však tímto nebudeme vůbec zabývat, ale každá stránka musí začínat právě takto.

Jako další musí být úplně celý kód stránky uzavřen do značek <html></html>, aby prohlížeč věděl, že celý obsah souboru je HTML kód

Poté se vždy na stránce vyskytuje značka <head>, která obsahuje informace o stránce, které slouží pro prohlížeč, dále značka <body>, obsahující data, které se zobrazí uživateli.

O značce <head> se nebudeme bavit, tuto část prostě zkopírujete podle šablony v sekci Pro vývojáře. Jediná pro vás zajímavá značka uvnitř značky <head> bude značka <title>, obsahující text, který se zobrazí nahoře, na kartě stránky v prohlížeči. Pro tuto stránku je to: <title>BATCH - PRO ZAČÁTEČNÍKY</title>. Nyní se však budeme bavit pouze o značce <body>

Horní navigační panel také pouze zkopírujete ze šablony, budete upravovat pouze tu část, která říká "Sem napište kód stránky".

Nejdůležitější značkou, kterou budete používat, je již dříve zmíněná značka <p>, neboli odstavec (paragraph). Cokoliv bude uvnitř této značky se zobrazí stejně jako text, který čtete právě teď.

Pokud budete chtít text upravovat (do značky lze vložit i další značky), tak když text bude ve značce <b>, tak se zobrazí tučně (bold) a když bude ve značce <i>, tak se zobrazí kurzívou (italic)

Když budete chtít odřádkovat (enter), tak napíšete značku <br>, která neobsahuje ukončovací značku! (žádné </br> neexistuje)

Například, když do souboru napíšete:


    <!DOCTYPE html>
    <html>
    <head></head>
    <body>
        <p>Tohle je text</p>
        <p><b>Tohle je tučný text</b>,
        <i>tento text je kurzívou</i> a
        <b><i>tento text je tučnou kurzívou</i></b></p>
    </body>
    </html>

tak se to zobrazí takovýmto způsobem:


Tohle je text

Tohle je tučný text, tento text je kurzívou a tento text je tučnou kurzívou


Psaní zdrojového kódu:

Veškeré nastavení zobrazování je už hotové, část v <head>, kde je napsané: <link rel="stylesheet" type="text/css" href="styles.css"> nastaví veškeré úpravy textu, pozadí, barev, atd. Pokud tedy napíšete pouze text ve značce <p>, nemusíte se už nijak starat o jeho zobrazení (Můžete si vyzkoušet, že když tento řádek z části <head> vymažete, tak veškeré grafické úpravy zmizí).

Kdybyste chtěli napsat nadpis, tak ten se píše pomocí značky <h[číslo]>, kde můžete dosadit čísla od 1 do 6, podle velikosti nadpisu (h1 je největší a h6 nejmenší). Zde mám zatím nastavené akorát nadpisy do h4.

Pokud chcete napsat zdrojový kód nějakého souboru, tak daný kód uzavřete do značek <pre> a <code>. Nemusíme si dopodrobna vysvětlovat, co tyto značky znamenají, ale prostě jsem to tak nastavil. Značka <pre> znamená, že se text zobrazí tak, jak je napsaný (např. značka <p> bude ignorovat mezery na začátku řádku a odřádkování) a značka <code> vypíše text tzv. strojovým písmem, tedy písmem, kde je každé písmeno stejně široké. Tohle se pro psaní zdrojových kódů často používá.

Pokud text uzavřete do <pre>, tak vám to i automaticky vyplní světlé pozadí za text, takže každý bude vědět, že se jedná o kód programu.

Když tedy napíšete kód, který vypadá takto:


    <!DOCTYPE html>
    <html>
    <head></head>
    <body>
        <h4>Příkaz IF</h4>
        <p>Kód za příkazem <b>if</b> se spustí jen tehdy, kdy platí podmínka</p>
        <p>Obecný zápis je: if <b>[Podmínka]</b> <i>[Příkaz]</i><br>např.:</p>
        <pre><code>
            @echo off
            set /a a = 5
            if <b>%a% == 5</b> <i>echo Text</i>
    
            set /a a = 10
            if <b>%a% == 5</b> <i>echo Text2</i>
            pause
        </code></pre>
        <p>Vypíše se pouze "Text", protože u "Text2" podmínka neplatí</p>
    
    </body>
    </html>

Tak se to zobrazí takto:


Příkaz IF

Kód za příkazem if se spustí jen tehdy, kdy platí podmínka

Obecný zápis je: if [Podmínka] [Příkaz]
např.:


        @echo off
        set /a a = 5
        if %a% == 5 echo Text
        
        set /a a = 10
        if %a% == 5 echo Text2
        pause

Vypíše se pouze "Text", protože u "Text2" podmínka neplatí


Vytváření seznamů

Pokud chcete vypsat seznam položek, tak existují dva různé seznamy, číslovaný a nečíslovaný. Jejich značky jsou <ul> pro nečíslovaný (odrážkový) seznam (unordered list) a <ol> pro číslovaný seznam (ordered list). Číslovaný seznam se používá v případě, že záleží na pořadí položek, pokud na pořadí nezáleží, používá se nečíslovaný seznam.

Každá položka jakéhokoliv seznamu se uzavírá do značek <li>

Takže např.:


    <!DOCTYPE html>
    <html>
    <head></head>
    <body>
        <p>Verze Windows:</p>
        <ul>
            <li>Windows XP</li>
            <li>Windows 95</li>
            <li>Windows 11</li>
            <li>Windows 7</li>
        </ul>

        <p>Verze Windows chronologicky:</p>
        <ol>
            <li>Windows 95</li>
            <li>Windows XP</li>
            <li>Windows 7</li>
            <li>Windows 11</li>
        </ol>
        
    </body>
    </html>

Tento kód se zobrazí takto:


Verze Windows:

  • Windows XP
  • Windows 95
  • Windows 11
  • Windows 7

Verze Windows chronologicky:

  1. Windows 95
  2. Windows XP
  3. Windows 7
  4. Windows 11

Jak vidím, tak jsem nastavil obří velikost písma u seznamů, když jsem vymýšlel nadpisy. Kdybyste to chtěli změnit, dovnitř značky <ol> nebo <ul> připište text class="mensi-pismo", takže to bude vypadat takto: <ul class="mensi-pismo">. Tohle zmenší velikost písma v seznamu na polovinu, teď se však nebudeme zabývat tím, co ten zápis znamená, je to tak prostě nastavené.

Kdybych nyní napsal tento kód znovu, ale s menším písmem, tak to bude vypadat takto:


    <!DOCTYPE html>
    <html>
    <head></head>
    <body>
        <p>Verze Windows:</p>
        <ul class="mensi-pismo">
            <li>Windows XP</li>
            <li>Windows 95</li>
            <li>Windows 11</li>
            <li>Windows 7</li>
        </ul>
    
        <p>Verze Windows chronologicky:</p>
        <ol class="mensi-pismo">
            <li>Windows 95</li>
            <li>Windows XP</li>
            <li>Windows 7</li>
            <li>Windows 11</li>
        </ol>
    
    </body>
    </html>

Tak se tento kód zobrazí takto:


Verze Windows:

  • Windows XP
  • Windows 95
  • Windows 11
  • Windows 7

Verze Windows chronologicky:

  1. Windows 95
  2. Windows XP
  3. Windows 7
  4. Windows 11
